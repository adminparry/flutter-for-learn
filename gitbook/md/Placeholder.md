# Placeholder

创建绘制框的小部件。

##### 继承
``` dart
class Placeholder extends StatelessWidget
```
##### 构造

``` dart
const Placeholder({
    Key key,
    this.color = const Color(0xFF455A64), // Blue Grey 700
    this.strokeWidth = 2.0,
    this.fallbackWidth = 400.0,
    this.fallbackHeight = 400.0,
  }) : super(key: key);
```
